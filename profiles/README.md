## Redefining the dev experience

Vantris makes building full-stack apps feel instant again — powered by
[h3](https://h3.unjs.io), [esbuild](https://esbuild.github.io) and
[rolldown](https://rolldown.rs).

<table>
  <tr>
    <td align="center" width="25%">
      <h3>⚡ Instant Server Start</h3>
      <p>On-demand transforms over native ESM, with fast dependency pre-bundling.</p>
    </td>
    <td align="center" width="25%">
      <h3>🔥 Lightning Fast HMR</h3>
      <p>CSS, React Fast Refresh and Vue SFC hot-reload — with a full-reload fallback.</p>
    </td>
    <td align="center" width="25%">
      <h3>🧩 Rich Features Out of the Box</h3>
      <p>TypeScript, JSX, <code>.vue</code>, env vars, server routes — extend with modules.</p>
    </td>
    <td align="center" width="25%">
      <h3>📦 Optimized Build</h3>
      <p>Production-ready hashed assets and a runnable Node server via rolldown.</p>
    </td>
  </tr>
</table>

<br>

## A shared foundation to build upon

<table>
  <tr>
    <td align="center" width="33%">
      <h3>🔌 Flexible Plugin System</h3>
      <p>Rollup-compatible plugins and Nuxt-style modules for dev and build.</p>
    </td>
    <td align="center" width="33%">
      <h3>🖥️ Server Routes</h3>
      <p>A <code>server/</code> folder + <code>useServer()</code> for backend APIs alongside your app.</p>
    </td>
    <td align="center" width="33%">
      <h3>⚛️ React & Vue</h3>
      <p>Optional <code>@vantris/react</code> and <code>@vantris/vue</code> modules — pick your UI stack.</p>
    </td>
  </tr>
</table>
